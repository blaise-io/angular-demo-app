# Introduction to Angular

This is an example app that demos how to do:

 - Modules and dependencies
 - Dependency injection
 - Routing
 - Templating
 - Controllers
 - Providers
 - Resources
 - Filters
 - Form validation


## Running

 - Make sure Python 2.7 is installed on your system
 - `./server.py`
 - Open `http://127.0.0.1:8000` in a browser
